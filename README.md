# Q1 - Counter Screen

## Description
This app displays a counter starting at 0 with two buttons:
- Increment
- Reset

## Features
- State is managed using mutableStateOf
- rememberSaveable is used to preserve state across screen rotation
- UI updates automatically through recomposition

## How to Run
Open the project in Android Studio and run on emulator or device.

## AI Usage
I used ChatGPT to review the implementation and confirm correct usage of rememberSaveable and state handling, and logistic in the specific methods.
